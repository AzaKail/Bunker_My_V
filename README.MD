# Bunker Game (Web App)

## Описание
Веб-реализация настольной игры "Бункер" с мультиплеерной логикой.

## Функционал
- регистрация пользователей (логин/пароль)
- создание и управление игровой сессией
- игровая логика (раунды, роли и т.д.)
- взаимодействие игроков

## Технологии
- Python (backend)
- HTML/CSS/JS (frontend)
- (если есть: Flask/FastAPI)

## Как запустить
uvicorn main:app --reload --port 8000
# затем открыть http://localhost:8000
